Progetti Evidenziati

Una selezione del mio lavoro accademico ed extra-curricolare

Machine Learning / 3D Object Detection

DiffuBox: Refining 3D Object Detection

Sviluppo e implementazione di un modello di diffusione point-based per l'affinamento di bounding box 3D LiDAR.

  • Condizionamento del modello su point clouds LiDAR per calcolare posizionamento, scala e rotazione degli orizzonti d'interesse.
  • Superamento dei limiti legati al dominio shift (cambio di sensori/condizioni esterne).
Computational Science / Statistica

Bayesian Image Analysis in Fourier Space

Framework Bayesiano in Python nello spazio di Fourier per la restorazione ad alta accuratezza di immagini microscopiche.

  • Utilizzo di ottimizzatori MAP e campionamento MCMC (Markov Chain Monte Carlo) parallelizzato per gestire l'inferenza spaziale.
  • Restorazione rapida di segnali affetti da forte rumore gaussiano e poissoniano.
Python Data Analysis

Quantitative Stock Market Analysis

Analisi quantitativa storica di indici di borsa ed andamenti azionari finanziari in Python.

  • Manipolazione e calcolo vettoriale di grossi dataset storici tramite NumPy e Pandas.
  • Calcolo dei profili di rischio, volatilità storica, medie mobili esponenziali e portafogli ottimali.
Tesi Triennale / Crittografia

Fondamenti matematici della Crittografia

Tesi di laurea incentrata sulla transizione matematica dai crittosistemi classici agli standard di sicurezza quantistici.

  • Analisi algebrica dettagliata dell'algoritmo asimmetrico RSA e basi di aritmetica modulare.
  • Studio delle risposte difensive a minacce derivanti dall'Algoritmo di Shor tramite protocolli QKD (BB84).
